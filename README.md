# Chatbot-ON-OFF-LED-IOT

## Overview

This project integrates Firebase, React Native, Arduino, and ESP32 to control an LED using a chatbot interface.

## Technologies Used

- **Firebase**
- **React Native**
- **Arduino**
- **ESP32**
